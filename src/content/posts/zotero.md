# Introduction

可以用来管理文献的工具有很多,比如 Mendeley, EndNote, NoteExpress 等。但是，经过相当长一段时间的摸索，我发现最好用的还是 Zotero。

Zotero至少有如下优点：

1. 开源，更新活跃
2. 丰富的插件
3. 强大的检索功能
4. 支持多级文件夹

# Plugin

- [ZotFile](http://zotfile.com/)：管理附件。
- [Better BibTeX](https://retorque.re/zotero-better-bibtex/)：结合LaTeX一起使用。
- [Zutilo](https://github.com/wshanks/Zutilo)：实现一些辅助功能，比如删除标签、拷贝URL等。
- [zotero-shortdoi](https://github.com/bwiernik/zotero-shortdoi)：管理文献DOI。


#  FAQ

- ***How can I see what collections my item is in?***

To see all the collections an item is in, select the item and then hold down the Option key (macOS), Control key (Windows), or Alt key (Linux). This will highlight all collections that contain the selected item. [[source](https://www.zotero.org/support/kb/collections_containing_an_item)]

- ***How can I move an item to another collection?***

To move items between collections, hold down Cmd (Mac) or Shift (Windows/Linux) when dragging items to the new collection.
