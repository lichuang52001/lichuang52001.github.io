# 如何安装软件

```python
yum install xxxx -y    # xxxx为软件名
```

# 查看软件版本

```python
xxxx -V    # xxxx为软件名
```

# Linux如何修改SSH登录密码

```python
# 登陆SSH控制台，然后输入下面指令：
passwd user    # 注：这里的user是你的SSH用户名，比如passwd root
# 输入密码
```
