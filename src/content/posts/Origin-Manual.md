>
> It is produced by OriginLab Corporation, and runs on Microsoft Windows.

# Function plot

已知函数表达式，使用该功能可以直接绘图。

路径：File --> New --> Function plot

# 2D Plot

## 双y轴

1. 准备好数据，比如一列x，两列y（分别对应左右y轴）
2. 选择x和第一列y，默认作图，默认y轴在左侧
3. 在图中任意位置单击右键 --> "New Layer(Axes)" --> "Right-Y"
4. 确认选中 2 图层，菜单栏选择 "Graph" --> "Layer Contents..." --> 导入数据，确认即可。
